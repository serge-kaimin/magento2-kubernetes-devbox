Usage: 
    devbox instance [NAME] [OPTIONS]
    bin/instance [NAME] [OPTIONS]

  Control instances on Kubrenetes

Options:
  -h, --help    - Help 
  -v, --verbose - verbose output
  -all          - do comand on all available instances

Management Commands:
    list        - list instances
    install     - install instance
    start       - start dev environment
    status      - show status of the dev environment 
    stop        - stop dev environment and all of the components
    upgrade     - upgrade instance
    delete      - delete instance
